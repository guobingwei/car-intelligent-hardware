# car-intelligent-hardware

为一个小朋友做的毕业设计，大概功能一个车载传感器系统。硬件部分是一个传感器，试试上报车辆的位置和温度，后台不断接受这些数据，然后在运营后台大屏上实时展示。

## 前端部分
>* 实时数据展示页面，采用`websocket`技术
>* 历史数据列表查询


## 后端部分
>* 接收传感器的数据，进行存储
>* 提供历史数据查询接口
>* `websocket 技术，实时推送数据给前端展示
